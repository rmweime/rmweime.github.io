Play the Classic 2048 Game

Game Overview

2048 is a fun and addictive puzzle game where the objective is to slide numbered tiles on a grid to combine them and create a tile with the number 2048.

How to Play

Use the arrow keys (or swipe on mobile) to move the tiles. When two tiles with the same number touch, they merge into one. The game ends when there are no more valid moves left!

  • Use your arrow keys to move tiles.
  • Combine matching numbers.
  • Reach the 2048 tile to win.
  • Keep going to beat your high score!
